# Fund Interest Calculator

## Description
The purpose of this project was to build a functional fund interest calculator that allows the user to compare and contrast certain investment funds. This project does not offer financial advice and is hypothetical only. 

## Main Features Implemented
- JavaScript Navigation 
- Search, Filter, and Sorting Controls
- Simple and Compound-interest Calculations
- Recurring Deposits
- Monthly, Quarterly, or Annually Compounding
- Live Result Without Reloading Page
- Compare with up to Four-Funds
- Calculation History Saved with `localStorage`
- Add, Load, Delete, and Clear User Saved Calculations
- Validated Fund-Request Form 
- Locally Saved
- Clean Mobile Navigation

## Languages Used
- HTML5
- CSS3
- Vanilla JavaScript
- Information stored on Browser `localStorage`
